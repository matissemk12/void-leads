# Void Leads

## Product
A lead management tool for Void Web Agency. Scrapes business contact info from URLs or via Brave Search, stores leads locally, and enables web design sales outreach including automated email mockup generation.

## Register
product (design serves the tool — UI clarity over brand expression)

## Stack
Vanilla HTML/CSS/JS frontend, Vercel serverless API routes (Node.js), localStorage for data persistence.

## Users
Solo operator (Matisse) doing cold outreach to sell website redesigns ($1,500 deposit, $1,500 on delivery, $500/mo recurring) to small home service businesses (junk removal, garage cleaning, etc.)

## Design direction
Dark UI, minimal chrome, functional over decorative. Existing palette: near-black bg (#0a0c11), card surfaces (#151926), blue accent (#6e8bff), teal secondary (#21d3b8), neutral text (#eef0f6, #9aa3ba). 8px spacing grid.
